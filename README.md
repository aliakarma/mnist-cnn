# 🧠 Handwritten Digit Recognition with CNN

This project implements a **Convolutional Neural Network (CNN)** to classify handwritten digits (0–9) using the **MNIST dataset**.  
It demonstrates deep learning basics in Computer Vision with TensorFlow/Keras.

---

## 🚀 Features
- CNN model with Conv2D, MaxPooling, Dropout layers
- Achieves ~99% accuracy on test data
- Training graphs included
- Pre-trained model saved in `results/`
